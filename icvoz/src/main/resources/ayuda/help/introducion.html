Introducion

Este Software llamado Icvoz (Interepretador de Comandos de Voz), nace de la necesidad de mejorar interacion humano-computador.

Icvoz pretende convertirse en una herramienta útil para interactuar con el computador mediante el habla, reduciendo la dependencia en el uso del teclado, mejorando la interacción con el computador y facilitando su uso a personas con alguna inhabilidad física.

Icvoz nace con el nombre de tesis "Sistema de reconocimiento de comandos de voz en español", con un objetivo principal de "Desarrollar un interpretador de comandos de voz utilizando redes neuronales artificiales, dando acceso a personas con inhabilidades físicas mejorando la interacción hombre-maquina".

Para que Icvoz funcione adecuadamente se requiere tecnicas avazadas en inteligencia artificial para su funiocnamiento como son las Redes Neuronales Artficiales, las cuales permiten reconcer la palabra pronuncida por el usuario, claro esta para que una Red Neutonal pueda funcionar bien se requiere un preprocesamiento del audio para lograr resultados adecuados.
