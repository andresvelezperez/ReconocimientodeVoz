Requerimientos del Computador

Los requerimientos para que IcVoz funcione son:

Hardware:

Se necesita un minimo de 512 MB de memoria Ram.

Se recomienda 50 MB de espacio libre en el disco duro.

Se recomienda que el PC posea un sistema de 1.6 GHz de velocidad real del procesador o superior.

Se necesita un (1) Microfono con supresion de ruido para funcionar correctamente.

 

Software:

Sistema Operativo: Windows XP o Superior. (Preferiblemente)

Sistema Operativo: Distribuciones Linux con Kernel 2.6 o Superior (Kde o Genome).

Se necesita la Maquina Virtual de Java 1.6 o superior para su funcionamiento.
